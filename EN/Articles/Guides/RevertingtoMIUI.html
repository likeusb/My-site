Reverting to MIUI

So you found an unfixable issue, found no use in a Custom ROM, or are trying to troubleshoot a problem and you want to start from scratch to fix possible issues. This guide will help you do just that!

Prerequisites

  • Phone, computer with ADB drivers, USB cable to connect the two
  • Mi Flash tool - Download: here
  • Your phone’s respective fastboot ROM file [CAUTION] You will lock the bootloader, DO NOT get the wrong file under any circumstances. This will irreversibly break your phone.

Steps

  • Extract the .tgz (or whatever else format ***FASTBOOT*** ROM you have) in advance
  • Connect computer to phone via USB
  • Open Mi Flash Tool
  • Put phone into fastboot
  • Download necessary drivers via Mi Flash Tool
  • Select the folder that you extracted from the archive
  • Select flash_all_lock in the bottom right
  • Click flash
  • As a sidenote, if there’s an error and it says that a folder is missing, that would be the log folder, make one in the Mi Flash Tool folder. Though with the OneDrive link I’ve included, you should be fine
  • Pray
  • Once it is done, reboot into MIUI

And congrats! You are now ready to lock the bootloader by going to Options>Advanced Options>Developer options (when enabled by clicking MIUI version in About phone numerous times) and locking it by toggling OEM Unlocking to off.
